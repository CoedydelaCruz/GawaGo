# GawaGo

Monorepo for the GawaGo platform:
- Web app: React + Bootstrap 5
- Backend API: Django
- Database style: MySQL
- Mobile app workspace

## Project Structure
- `frontend/` React web client
- `backend/` Django API and business logic
- `mobile/` Mobile app workspace
- `infra/` Infrastructure and DB bootstrap files
- `docs/` Architecture and API documentation
- `scripts/` Utility scripts
- `tests/` Cross-project test plans
